polyclonal package¶

polyclonal¶

Package for modeling mutational escape from polyclonal antibodies using deep mutational scanning experiments.

Importing this package imports the following objects into the package namespace:

  • Polyclonal

  • PolyclonalCollection

  • PolyclonalAverage

  • PolyclonalBootstrap

It also imports the following alphabets:

  • AAS

  • AAS_WITHSTOP

  • AAS_WITHGAP

  • AAS_WITHSTOP_WITHGAP
